午休的时候，同事问道：“去隔壁网点开过基金户没？”。

我说，去网点开户还得排队，多麻烦。上网银开吧，或者直接在基金公司的网站上买，费率更便宜。

最近刚好在玩基金，同事这么一问，下了班我就总结了一下购买基金有哪些渠道。

目前主流的基金购买渠道有这么四个：

  1. 银行网点代销
  2. 证券公司代销
  3. 基金公司直销
  4. 第三方销售渠道

不同的渠道，在便利性、费用、提供的服务上都有较大的差别。我们就逐一来看一下它们之间的差别在哪里，以及如何选择适合自己的渠道去购买基金。

购买基金会产生哪些费用？

首先来看看基金费用有哪些。

费用种类

基金费用一般包括两大类：

  • 申购/赎回等手续费

这类费用是在基金购买和赎回过程中发生费用。主要包括认购费、申购费、赎回费和基金转换费。

  • 管理费用

这一类是在基金管理过程中发生的费用。主要包括基金管理费（给基金公司和基金经理人）、基金托管费（给银行）、信息披露费等。

费用案例

如果你投资1万元购买1只股票型基金，一般申购费为1.5%，即150元。如果半年之后你的投资资产增值为11000元，那么赎回时你还需要付出赎回费用11000 × 0.5% = 55元。前后付出的操作成本（申购/赎回手续费）为150＋55=205元。

加上基金公司计提的管理费，一般股票基金每年为1.5%，托管银行每年提取0.25%的托管费等。你在一年内承担的管理费用在10000 × 1.75% = 175元。

成本总计：205 + 175 = 380元。（注：以上均是指费率无优惠的情况）

除了费用之外

以上就是购买基金产生的费用明细了。当然，除了费用之外，我们也要兼顾以下几项：

  1. 可认/申购时间
  2. 可选基金品种
  3. 网点数量
  4. 认购流程

综合考虑，选择适合自己的购买方式。

银行网点代销

基金卖得最多的地方就是各大银行了，这些银行堪称基金界的“7-11”。银行的客户渠道比较广、下单方便、可购买的基金类型较多，加上银行作为信用中介受到客户的广泛信任，所以在基金销售方面的市场占有率一直都比较高。

  • 认/申购费率：通过网上银行申购一般有8折左右的折扣。
  • 认/申购时间：周一至周五，银行营业时间段内；网上银行则是24小时开放。
  • 可选基金品种：一般为该银行托管及代销基金，投资者可事先查询想要的基金再到相关代销银行购买。
  • 网点数量：各渠道中销售网点最多的就是银行了。

投资者在银行购买基金，只需要到网点办理一个借记卡账户并开通购买基金的功能。可以通过柜台购买，也可以在网上银行购买。

银行网点适合所有人，特别是新手和年纪稍大的中老年基金投资者，他们往往更需要实体网点的可信任感和柜台业务员的建议来购买基金。

证券公司代销

除了银行之外，在证券公司也可以买到基金，但市场占有率没有银行那么高。
一般来说，证券公司代销大多数基金公司的产品，可选择面也很广泛。费率比银行低，但是网点比银行少。堪称基金界的“便利商店”。

  • 认/申购费率：通常为6折至9折不等。
  • 认/申购时间：各大券商营业部的营业时间内，与银行网点类似。
  • 可选基金品种：可供选择的基金种类、数量繁多，包括参与上市交易的封闭式基金、ETF、LOF基金。
  • 网点数量：券商营业部数量远不及银行，但比基金公司直销网点多。

投资者首次办理需要到证券公司开立股票账户或资金账户，然后通过券商软件进行购买。对比银行的基金销售来说，证券公司的客户经理具备更专业的投资能力，能够提供良好的分析建议。所以证券公司代销更适合工薪阶层或年轻白领以及券商的其他客户。

基金公司直销渠道

通过基金公司直销渠道购买基金，就相当于我们直接去工厂采购，所以能拿到“批发价”（低至四折的手续费）。

  • 认/申购费率：通常在4折至6折。
  • 认/申购时间：一般都是在基金公司网站24小时开放购买。
  • 可选基金品种：只销售该公司自己的基金产品。
  • 网点数量：较少。

我们可以在基金公司的网站在线开通基金账户，在网上就能实现认/申购、赎回等交易，不受时间地点的限制。如此购买的费率是最优惠的，购买也很方便，适合有较强专业能力、能对基金产品进行分析、能上网办理业务的投资者（当然，我们的目标就是逐步成为这样的专业投资者）。而直销柜台适合机构或资金量较大的个人客户。

直销渠道的缺点是：每家基金公司都只销售自己的基金产品。我们显然需要配置多支基金，这就得在多家基金公司的网站上开户，投资管理的操作成本就很高。

第三方销售渠道

除了以上三种，还有许多第三方平台汇聚各种基金的信息，能直接进行各基金的直销（链接到各基金公司网站），比如天天基金网、汇付天下等。他们就像基金界的“沃尔玛”：费率和基金公司直销一样优惠，更可以方便地在不同的基金公司产品间进行对比和挑选。

  • 认/申购费率：大部分能达到4折的优惠。
  • 认/申购时间：一般为网上认购，不受时间地点限制。
  • 可选基金品种：品种很多。
  • 网点数量：网点数量很少，大多数是通过网上形式销售，交易比较便捷。

适合人群：专户投资者、需要投资多只基金进行投资组合的人。

以上就是四种购买渠道的相关细节对比。我个人是在第三方网络代销平台购买，手续费都是4折。其实如果投资了一段时间，就会发现对于偏股型基金来说，选好买入时机和品种是最关键的，费率是个次要因素。而债券型基金中，有不少设置了没有申购费的B类或C类份额，这些份额在哪里买效果都是一样的了。

小白课堂

认购和申购是什么意思？？

基金首次募集期购买基金的行为称为认购；

在基金成立后购买基金的行为称为申购；

一般情况下，认购期购买基金的费率要比申购期购买优惠。认购期购买的基金一般要经过封闭期才能赎回，而申购的基金则是在申购成功后的第二个工作日就可赎回。

场内和场外基金

场内和场外的区别是以交易所能否直接交易来划分的。

场内基金是可以用股票账户按照买入和卖出方式操作的基金，可以t日买入确认，手续费按照股票交易收取。比如封闭式基金，ETF基金。

场外基金是采取申购和赎回的操作方式，t日申购，t+1日确认。也就是银行、证券公司的代销，基金公司的直销方式，也就是大家所熟悉的开放式基金。

二者的主要区别是，场内基金要开股票账户，场外基金不需要开股票户，还有就是手续费的区别，场外基金费用要高于场内。
